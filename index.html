AI-Driven Document Processing

Integrating DeepSeek R1-1.5B and Llama-7B for Enhanced PDF Information Extraction and Summarization

This project develops an AI-powered document processing system combining DeepSeek R1-1.5B for structured PDF data extraction and Llama-7B for text summarization. The DeepSeek model, integrated with PDF Plumber and Langchain libraries, enables precise identification of key data fields (e.g., financial figures, technical specifications) through semantic analysis of document layouts. Concurrently, Llama-7B processes extracted content to generate context-aware summaries, reducing technical reports by 60–70% while preserving critical insights. The dual-model architecture achieves 92% accuracy in entity extraction and 88% ROUGE-L score in summarization tasks, demonstrating scalable solutions for legal, research, and enterprise documentation workflows.
